🛒 تقدية – دليل التثبيت

3 طرق لتشغيل التطبيق على هاتفك — اختر الأسهل

1 GitHub Pages مجانيالأسهل
  1. اذهب إلى github.com وأنشئ حساباً مجانياً
  2. انقر New repository → اسمه: taqdiya → فعّل Public
  3. ارفع الملفات الأربعة: index.html, manifest.json, sw.js, ومجلد icons/
  4. اذهب إلى Settings → Pages → Source: main branch / root → Save
  5. رابطك سيكون: https://اسمك.github.io/taqdiya
  6. افتح الرابط على هاتفك في Safari (iOS) أو Chrome (Android)
  7. iOS: زر المشاركة ← "أضف إلى الشاشة الرئيسية"
  8. Android: النقاط الثلاث ← "تثبيت التطبيق" أو "أضف إلى الشاشة"
✓ التطبيق يعمل بدون إنترنت بعد أول تحميل | ✓ يحفظ بياناتك محلياً
2 Netlify Drop مجاني
  1. اضغط على مجلد taqdiya-pwa وانسخه كاملاً
  2. اذهب إلى app.netlify.com/drop
  3. اسحب مجلد taqdiya-pwa وأفلته في الصفحة
  4. في ثوانٍ ستحصل على رابط مثل: https://random-name.netlify.app
  5. افتحه على هاتفك وثبّته كـ PWA
✓ الأسرع — لا حاجة لحساب | ✓ يمكن ربطه بدومين خاص لاحقاً
3 تشغيل محلي عبر WiFi بدون إنترنت
  1. تأكد أن الكمبيوتر والهاتف على نفس الشبكة WiFi
  2. افتح Terminal / Command Prompt في مجلد taqdiya-pwa
  3. شغّل أحد الأوامر حسب نظامك:
python3 -m http.server 8080
npx serve . -p 8080
php -S 0.0.0.0:8080
  1. اعرف IP الكمبيوتر (ipconfig / ifconfig) مثلاً: 192.168.1.5
  2. افتح على هاتفك: http://192.168.1.5:8080
⚠️ يعمل فقط عندما الكمبيوتر شغّال وعلى نفس الشبكة — للتجربة فقط

📱 تثبيت على الهاتف (بعد فتح الرابط)
